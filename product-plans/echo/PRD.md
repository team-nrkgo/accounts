# Product Requirements Document: Echo.nrkgo.com

**Version:** 1.0 (MVP)
**Project Name:** Echo
**Parent Brand:** NRKGO

## 1. Executive Summary

Echo is a next-generation blogging SaaS built for the 2026 AEO (Answer Engine Optimization) era. It enables users to multiply their voice across the web by transforming keywords and URLs into high-authority articles that AI models (like ChatGPT and Perplexity) will cite as sources.

## 2. Target Audience

- **Solopreneurs:** Need high-quality content without the "blank page" struggle.
- **Niche Authorities:** Want their blogs to be the primary source for AI-driven search.
- **Content Marketers:** Moving from traditional SEO to AI-first AEO.

## 3. Core Features (MVP Scope)

### 3.1 The "Echo-Point" Generator (3 Entry Methods)

- **Manual Echo:** A blank, distraction-free editor for original writing.
- **Source Echo:** User provides a URL; AI parses the content and generates a new, unique draft.
- **Search Echo:** User provides a target keyword; AI researches the intent and generates an SEO/AEO-optimized draft.

### 3.2 The Unified Workspace

- **Tiptap Editor:** A single, block-style editor (like Notion) used for all drafts.
- **Status Workflow:** Support for Draft, Review, Scheduled, Published, and Trash.
- **Management:** Dashboard with a toggle between 'List View' (Rows) and 'Document View' (Visual Cards).

## 4. User Experience (UX)

- **Zen Editor:** No clutter. Right-hand sidebar for metadata (Slugs, Images, Categories).
- **Manual Control:** Big, green "Save" button (no auto-save for MVP to ensure user intent).
- **Subdomain Branding:** Hosted at {user-blog}.nrkgo.com.

## 5. Success Metrics

- **Time-to-Echo:** Reducing the time from idea to publish to under 5 minutes.
- **AEO Readability:** Content structure optimized for AI scrapers/crawlers.
